# chinesedict

## Overview

ChineseDict aims to be a fast Chinese-English dictionary. Its main goal is
providing instant-search to enable looking up words rapidly during discussion
or watching a video.

## Development

```bash
npm run dev
```

## Deploying

```bash
# Build for prod
npm run build

# Preview prod
npm run preview
```
